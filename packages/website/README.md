# allourthings.io

Astro static site for [allourthings.io](https://allourthings.io), deployed to Cloudflare Pages.

## Development

```bash
task website:dev     # start dev server at localhost:4321
task website:build   # build to dist/
task website:deploy  # build and deploy to Cloudflare Pages
```

Run all commands from the repo root.
